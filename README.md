## How to use this repo

1. Read the `lessons/` folder in order (level-01, level-02, ...)
2. Each lesson has: TL;DR, analogies, key concepts, annotated code, an exercise, and pitfalls
3. **Do the exercise** — reading is not learning, typing is
4. Each lesson is self-contained; you can skip ahead if you already know the basics
