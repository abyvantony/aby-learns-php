# Level 3: Conditionals: if, else, switch

**Track:** PHP · **Level:** 3/12 · **Difficulty:** `beginner`

📚 **Today's lesson** — published 2026-06-27

## TL;DR

Conditionals let your code make decisions. `if` runs code only when something is true. `else` runs code when it's false. `switch` handles many cases cleanly.

## Real-world analogies

- An `if` statement is a bouncer at a club. If you're on the list (condition is true), you go in (code runs). Otherwise, you're turned away.
- A `switch` is a vending machine — you press a button (match the value), and the matching item drops out.

## Key concepts

### `Boolean expressions`

Anything that evaluates to true or false: `$age >= 18`, `$name === 'Aby'`, `count($items) > 0`.

### ``elseif` vs `else if``

Same thing in most cases. `elseif` is one keyword; `else if` works in nested blocks. Pick one style and stick to it.

### `Truthy and falsy`

Only `false`, `0`, `0.0`, `"0"`, `""`, `null`, and empty arrays are falsy. EVERYTHING else is truthy, including `"false"` and `[]` (empty array in PHP 8+).

### `Ternary shorthand`

`$x = $cond ? $a : $b;` is a one-line if/else. PHP 7.4+ added `??` (null coalescing): `$x = $a ?? $b;` means "if $a is null, use $b".

## Code with comments

Every line has a comment. Read it slowly.

```php
<?php
$age = 20;
$hasLicense = true;

// Basic if/else
if ($age >= 18) {
    echo "You can vote.\n";
} else {
    echo "Too young to vote.\n";
}

// if/elseif/else chain
if ($age < 13) {
    echo "Child";
} elseif ($age < 20) {
    echo "Teenager";
} elseif ($age < 60) {
    echo "Adult";
} else {
    echo "Senior";
}

// Switch — cleaner when comparing one value to many options
$day = "Monday";
switch ($day) {
    case "Saturday":
    case "Sunday":
        echo "Weekend!";
        break;
    case "Monday":
        echo "Back to work.";
        break;
    default:
        echo "Weekday.";
}
// Don't forget `break` — without it, PHP falls through to the next case

// Ternary
$status = ($age >= 18) ? "adult" : "minor";

// Null coalescing (PHP 7+)
$username = $userInput ?? "guest";  // If $userInput is null, use "guest"

// Null coalescing assignment (PHP 7.4+)
$config['debug'] ??= false;  // Set to false only if not already set

// Match expression (PHP 8+) — strict switch, returns a value
$result = match (true) {
    $age < 13 => "child",
    $age < 20 => "teen",
    $age < 60 => "adult",
    default   => "senior",
};
```

## Try it yourself

Write a grade calculator: given a score 0-100, print A (90+), B (80+), C (70+), D (60+), F (below 60). Use a switch first, then rewrite with a match expression.

## Common pitfalls

- ⚠️ Using `=` (assignment) instead of `==` or `===` (comparison) is the #1 PHP bug.
- ⚠️ Forgetting `break` in a switch makes cases fall through to the next one.
- ⚠️ Truthy check on strings: `if ("0")` is FALSE because "0" is a falsy string. Use `if ($x !== "")` or `if (strlen($x) > 0)` instead.
