# Level 4: Loops: for, while, foreach

**Track:** PHP · **Level:** 4/12 · **Difficulty:** `beginner`

📚 **Today's lesson** — published 2026-06-28

## TL;DR

Loops repeat code. `for` is for counted iterations, `while` is for unknown counts, `foreach` is the easiest way to walk through arrays.

## Real-world analogies

- A `for` loop is a clock: start at 12, tick once per second, stop at 6.
- A `while` loop is washing dishes until the sink is empty — you don't know how many dishes there are.
- A `foreach` loop is a tour guide walking through a museum, pointing at each painting one at a time.

## Key concepts

### `Loop variable`

The counter in `for` (`$i`) or the current value in `foreach` (`$item`). It's a normal variable you can name anything.

### `Infinite loops`

If the condition never becomes false, the loop runs forever. Always make sure the loop makes progress toward ending.

### ``break` and `continue``

`break` exits the loop entirely. `continue` skips the rest of this iteration and goes to the next one.

### ``foreach` reference`

Use `&$value` to modify the original array element. Don't forget `unset($value)` afterward or weird things happen.

## Code with comments

Every line has a comment. Read it slowly.

```php
<?php
// for loop — when you know how many times to run
for ($i = 0; $i < 5; $i++) {
    echo "Count: $i\n";
}
// Output: Count: 0, Count: 1, ... Count: 4

// while loop — when you don't know how many iterations
$countdown = 10;
while ($countdown > 0) {
    echo "$countdown... ";
    $countdown--;
}
echo "Lift off!\n";

// do-while — runs at least once, then checks
$roll = 0;
do {
    $roll = rand(1, 6);
    echo "Rolled: $roll\n";
} while ($roll !== 6);

// foreach — the easy way to walk arrays
$fruits = ["apple", "banana", "cherry"];

// Get just values
foreach ($fruits as $fruit) {
    echo "$fruit\n";
}

// Get both key and value
$prices = ["apple" => 1.50, "banana" => 0.50, "cherry" => 3.00];
foreach ($prices as $fruit => $price) {
    echo "$fruit costs \$$price\n";
}

// Modify array values with reference (note the &)
$numbers = [1, 2, 3];
foreach ($numbers as &$n) {
    $n *= 10;  // Multiply each by 10
}
unset($n);  // Critical: break the reference, otherwise $n leaks into next loop
print_r($numbers);  // [10, 20, 30]

// break and continue
for ($i = 0; $i < 10; $i++) {
    if ($i === 3) continue;   // Skip 3
    if ($i === 7) break;       // Stop at 7
    echo "$i ";
}
// Output: 0 1 2 4 5 6
```

## Try it yourself

Create an array of 10 numbers. Use foreach to find the largest. Then use a for loop to print the multiplication table (1-10) for any number the user picks.

## Common pitfalls

- ⚠️ Off-by-one errors: `< 10` runs 0-9, `<= 10` runs 0-10. Pick the right one for what you mean.
- ⚠️ Using `foreach ($arr as $item)` and trying to modify `$item` — the original array doesn't change. Use `&$item`.
- ⚠️ Forgetting to `unset($value)` after a reference foreach — the variable stays bound to the last array element.
